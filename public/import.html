CSV Format Guidelines:
  • id_number: Unique alphanumeric ID (8-12 characters)
  • full_name: Complete name
  • date_of_birth: Format: DD/MM/YYYY (e.g., 15/05/1980)
  • address: Barangay name only (e.g., "Bayanan I", not "Barangay Bayanan I")
  • sex: Male or Female
  • image: URL or file path (optional)
  • signature: URL or file path (optional)
  • rsbsa: RSBSA number (optional)
  • contact_number: Phone number (optional)
  • Activity categories (use 1 for Yes, 0 for No):
    • boat_owneroperator
    • capture_fishing
    • gleaning
    • vendor
    • fish_processing
    • aquaculture
